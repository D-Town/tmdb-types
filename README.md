Included are Types for:

- [ ] Authentification
- [ ] Certification
- [ ] Changes
- [ ] Collections
- [ ] Compaines
- [ ] Configuration
- [ ] Credits
- [ ] Discover
- [ ] Find
- [ ] Genres
- [ ] Guest Sessions
- [ ] Keywords
- [ ] Lists
- [ ] Movies
- [ ] Networks
- [ ] Trending
- [ ] People
- [ ] Reviews
- [ ] Search
- [ ] TV
- [ ] TV Seaons
- [ ] TV Episodes
- [ ] TV Episode Groups
